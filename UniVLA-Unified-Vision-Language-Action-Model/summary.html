UniVLA: Unified
Vision-Language-Action

想让同一个大模型同时看懂视频、回答问题、输出机器人动作：与其给 VLM 外挂一个视觉编码器再接动作头，不如把视觉、语言、动作全变成离散 token，在同一个自回归 Transformer 里联合建模——再插一个世界模型后训练阶段，用海量无动作标注的机器人视频学时序因果动态，然后再微调出策略。

💡速记层 · 思想 / 逻辑 / 方法

默认读完就走的一层——只讲思想、逻辑、方法；效果只作验证。要核对原文/钻细节再看下面的「深读层」。

💡 一句话思想
现有 VLA 给 VLM 外挂视觉编码器再接动作输出，是个晚期融合的语言中心范式，天然丢失视频里的时序因果结构。UniVLA 的答案：把视觉图像、自然语言、机器人动作全部 tokenize 成同一份词表里的离散 token，让同一个 8.5B 自回归 Transformer 用"下一 token 预测"统一学习——再插一个世界模型后训练阶段，用大规模无动作标注机器人视频学因果动态，最后微调即可高效学出策略。泛化靠时序理解的预训练迁移，不靠蛮力堆动作数据。
🔗 逻辑链 · 为什么这个想法成立
  1. 主流 VLA（RT-2、OpenVLA、π0）让 ViT 编码图像再接语言模型——这种晚期融合限制了跨模态深度耦合，且无法利用视频里的时序与因果依赖。
  2. 真正的统一：把图像用 VQ tokenizer 离散化，把动作用 FAST（DCT）离散化，与语言共享词表 → 一切都是 token，next-token prediction 统一一切。
  3. 但光有架构还不够：动作数据稀缺，视频数据海量。中间插一个世界模型后训练——只用视频（无需动作标注）、以视觉 token 为监督，学环境动态 P(s_{t+1}|s_t, instruction)。
  4. 世界模型学到的时序因果表征可大幅迁移到下游策略微调：用 10% 数据即可超过全数据 GR-1；2k 步收敛即超过 8k 步无后训练基线。
  5. 同一框架自然支持多模态输出——spatial grounding（文本）、video prediction（视觉）、action（动作）都是 next-token prediction 的不同 loss mask。
🧩 方法骨架 · 三个关键设计决策
①统一自回归建模：图像用 VQ tokenizer（8× 空间压缩），动作用 FAST（DCT 离散化），加 boi/eoi/boa/eoa 特殊分隔符，与语言 token 拼成同一序列——同一个 8.5B Transformer（基于 Emu3）做 next-token 预测。无独立 ViT encoder。
②世界模型后训练：用 622K 机器人视频（RT-1、BridgeV2、DROID、CALVIN 等）在 instruction + video 序列上训练，只对视觉 token 计算 loss，不需要动作标注。对比消融：纯动作预测后训练会负迁移（-0.94 CALVIN avg）；世界模型后训练提升最大（+3.15）。
③两帧交错策略微调：以 {lang, obs_t, action_t, obs_{t+1}, action_{t+1}, ...} 交错序列微调，action chunk 大小 10（CALVIN/LIBERO）/ 5（SimplerEnv）；引入 1 帧历史观测窗口效果最优（CALVIN 4.26→4.61）；同时保留视觉预测 loss（权重 0.5）可再涨约 0.2 avg。
✓ 效果 = 验证（不是重点）

三项仿真基准全部刷新 SOTA；世界模型消融证明时序因果理解是性能跃升的核心——这反过来支持"统一 token 序列 + 世界模型预训练"的思想。

◆核心观点 · 证据

每条 = 中文论点 + 📍页/节 + 🔤逐字原文(Ctrl-F 锚点) + 🀄译文 + 💡解读。点开原文即可最短路径核对。

观点1 真问题：现有 VLA 的视觉-语言-动作是"晚期融合"范式，丢失了视频里的时序与因果结构，也无法利用大规模无动作视频数据。
📍 p.2 · §1 Introduction
most existing VLA approaches follow a language-centric paradigm: visual observations are first projected into a semantic space, and action policies are subsequently derived based on these representations. This late-fusion strategy, while beneficial for semantic understanding and generalization, limits the formation of deeply coupled cross-modal representations and impedes the learning of temporal and causal dependencies across the perception-action loop.
现有 VLA 方法大多遵循以语言为中心的范式：视觉观测首先被投影到语义空间，然后据此推导动作策略。这种晚期融合策略虽有利于语义理解和泛化，却限制了跨模态深度耦合表征的形成，并阻碍了感知-动作循环中时序和因果依赖关系的学习。
这是 UniVLA 的立论基础：既有 VLM+ViT+动作头的架构不是"真统一"，是拼凑。
观点2 核心方案：把视觉、语言、动作全部离散化成同一词表内的 token，用单一自回归 Transformer 做 next-token prediction——这是真正的"统一"建模。
📍 p.1 · Abstract
we present UniVLA, a unified and native multimodal VLA model that autoregressively models vision, language, and action signals as discrete token sequences. This formulation enables flexible multimodal tasks learning, particularly from large-scale video data.
我们提出 UniVLA，一个统一的原生多模态 VLA 模型，将视觉、语言和动作信号自回归地建模为离散 token 序列。这一表述支持灵活的多模态任务学习，尤其擅长从大规模视频数据中学习。
关键词：native multimodal（原生，不是拼接）、discrete token sequences（统一词表）、autoregressively（同一个 next-token 目标）。
📍 p.4 · §3.1 Unified Multimodal Model
our method unifies language, vision, and action modalities by converting each into discrete tokens and concatenating them into a single multimodal sequence L. Specifically, Lt, Lv, and La denote the discrete token sequences for language, vision, and action, respectively, all drawn from a shared vocabulary.
我们的方法通过将语言、视觉和动作模态各自转换为离散 token 并拼接为单一多模态序列 L，从而统一建模。Lt、Lv、La 分别表示语言、视觉和动作的离散 token 序列，均来自共享词表。
共享词表是关键：所有 token 的 loss 函数都是 cross-entropy，不同 token 类型靠 loss mask 区分。
观点3 世界模型后训练是关键"催化剂"：用 instruction + video 序列（无需动作标注）预测未来视觉 token，让模型学到时序因果动态——这一步的增益远超其他后训练策略。
📍 p.5 · §3.2 World Model (Post-train)
we use the loss computed solely from the vision tokens as the supervisory signal, enabling the model to generate visual predictions conditioned on the given instruction and observed state.
我们仅以视觉 token 上的 loss 作为监督信号，使模型能够在给定指令和观测状态下生成视觉预测。
无动作标注 = 622K 视频全可用；监督目标是"预测下一帧"，强迫模型理解"做了这个指令，世界会怎么变"。
📍 p.7-8 · §4.4 Table 4
the world model post-training approach yields the most substantial gains, enhancing both generalization and long-horizon planning capabilities. A comparison with text-to-image (T2I) training emphasizes the importance of modeling temporal dynamics in video data, while contrasting with video-only training highlights the essential role of textual guidance in state transitions.
世界模型后训练方法带来最大增益，同时提升泛化和长程规划能力。与 T2I 训练的对比强调了在视频数据中建模时序动态的重要性；与纯视频训练的对比则凸显了文字指导在状态转移中的关键作用。
必须同时有 text（任务指令）和 video（时序观测），两者缺一都不行——这正是"世界模型"而非"视频生成"的本质。
观点4 世界模型后训练带来显著的数据效率和训练效率提升：10% 数据即超过以往全数据方法；2k 步已超无后训练 8k 步——迁移是真实且大幅的。
📍 p.8 · §4.4 Data and Training Efficiency
on the CALVIN benchmark (Table 5a), our method achieves higher success rates using only 10% of the fine-tuning data, outperforming prior approaches such as GR-1 and RoboVLMs.
在 CALVIN 基准（表 5a）中，我们的方法仅用 10% 的微调数据就取得了更高的成功率，超越了 GR-1 和 RoboVLMs 等先前方法。
10% 数据下 UniVLA = 3.19 avg，GR-1 全数据 = 2.00；世界模型后训练实质上给策略微调提供了"免费的"时序理解基础。
观点5 在 LIBERO、CALVIN、SimplerEnv 三项仿真基准上全面刷新 SOTA，尤其长程任务（LIBERO-Long）大幅领先——证明统一 token 框架 + 世界模型后训练在长程推理上的优越性。
📍 p.6 · §4.3 Main Results (LIBERO)
UniVLA achieves the best overall performance across all LIBERO benchmark suites, with particularly significant gains on long-horizon tasks—improving the previous state of the art from 69.0% to 94.0%.
UniVLA 在所有 LIBERO 基准套件上均取得最佳整体表现，在长程任务上尤为显著——将前 SOTA 从 69.0% 提升至 94.0%。
LIBERO-Long 的 +25pp 是该论文最醒目的单项数字；长程任务需要跨时间步的规划，恰好是世界模型后训练增益最大的地方。
📍 p.6 · §4.3 Main Results (SimplerEnv)
Our approach demonstrates a significant improvement over prior methods, raising the average success rate from 42.7% to 69.8%.
我们的方法相比先前方法展示了显著提升，将平均成功率从 42.7% 提升至 69.8%。
SimplerEnv 用真实机器人数据的仿真评测，代表 sim2real 泛化能力，+27pp 说明统一 token 学到了可迁移的视觉-动作对应。
观点6 同一模型天然支持多模态输出——spatial grounding（输出 bounding box）、future frame prediction（输出视觉 token）、action prediction（输出动作 token）——靠不同的 loss mask 切换，不需要多个专用模型。
📍 p.9 · §4.5 Multimodal Capability
This design enables policy learning for embodied control, spatial reasoning through language output, and future state prediction via visual output, highlighting the model's capacity for generalizable multimodal understanding.
这种设计使得模型能够用于具身控制的策略学习、通过语言输出进行空间推理，以及通过视觉输出进行未来状态预测，彰显了模型可泛化的多模态理解能力。
这不只是"附赠功能"——grounding 和 visual prediction 能力正是世界模型后训练的副产品，也解释了为何后训练能迁移到策略微调。
观点7 框架跨域适应性强：真机 ALOHA 双臂实验（AgileX Cobot Magic V2.0，8 类任务）和 NAVSIM 自动驾驶基准均证明，同一框架无需大改即可迁移到新平台，且世界模型后训练在真机端同样有效。
📍 p.17 · Appendix B Real-Robot Experiments
Interestingly, this post-training provides substantial benefits even when transferring to real-robot execution.
有趣的是，即使在迁移到真实机器人执行时，该后训练依然带来了显著收益。
真机验证了世界模型后训练不只是仿真技巧——它学到的时序因果知识在真实物理环境中也有用。

⚖批判 · 评级

✅ 值得读的理由

架构思路干净：真正的"统一 token"而非拼凑；世界模型后训练这个"中间阶段"的设计既有理论支撑（MDP 世界模型），又有详细的消融验证（Table 4 各策略对比）。LIBERO-Long +25pp 和 SimplerEnv +27pp 是量级上的领先，不是刷榜小数点。代码权重已开源、数据集全公开，可复现性强。ICLR 2026 接收背书质量。对 Track A 研究者：「无动作标注视频 → 世界模型 → 高效策略迁移」这条路线是直接可借鉴的训练配方。

🔍 主要保留

① 仅仿真评测为主：三项 SOTA 结果均来自仿真基准；真机实验（ALOHA）仅定性展示，未提供标准化成功率对比。② 模型体量大：8.5B 参数，后训练需要 32 × A100 × 4-5 天，实际部署门槛高。③ 无独立 ViT encoder 是把双刃剑：统一 token 丢失了空间精细结构（VQ tokenizer 8× 压缩），高精度操作（微米级放置）可能受限。④ 论文自承后训练规模化研究尚处早期，与 RL 范式的结合留作未来工作。⑤ 基于 Emu3 的 VQ vision tokenizer 在运动模糊/快速动作场景下的重建质量未有讨论。

评级：Important  在 VLA 统一建模路线上，UniVLA 是目前最系统的实现之一，尤其"世界模型后训练"对数据效率的大幅提升是重要发现，但仍以仿真为主，真机大规模验证有待跟进。
